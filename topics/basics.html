Basics

A GameObject is the fundamental object in Unity. Everything in your scene is a GameObject.

GameObject myObject = new GameObject("My Object");

To add a component, like a Rigidbody, to a GameObject, use the AddComponent method.

myObject.AddComponent<Rigidbody>();

You can set the position of a GameObject using the transform.position property.

myObject.transform.position = new Vector3(0, 1, 0);
