Moving, Rotating, and Scaling

Moving a GameObject

To move a GameObject to a specific position, set its transform.position.

myObject.transform.position = new Vector3(1, 2, 3);

To translate a GameObject by a vector, use the Translate method. This is useful for moving the object relative to its current position.

myObject.transform.Translate(Vector3.forward * Time.deltaTime);

Rotating a GameObject

To set the rotation of a GameObject, use the transform.rotation property with Quaternion.Euler.

myObject.transform.rotation = Quaternion.Euler(45, 90, 0);

To rotate a GameObject by angles, use the Rotate method. This method rotates the GameObject around its local axes.

myObject.transform.Rotate(new Vector3(0, 30, 0) * Time.deltaTime);

Scaling a GameObject

To set the scale of a GameObject, use the transform.localScale property.

myObject.transform.localScale = new Vector3(2, 2, 2);

To scale a GameObject by a factor, multiply its transform.localScale. This can be used to uniformly or non-uniformly scale the GameObject.

myObject.transform.localScale *= 1.1f;

Example Script

Here is a complete example script that moves, rotates, and scales a GameObject over time.

using UnityEngine;

public class MoveRotateScale : MonoBehaviour
{
    void Update()
    {
        // Move the GameObject forward continuously
        transform.Translate(Vector3.forward * Time.deltaTime);
        
        // Rotate the GameObject around the y-axis
        transform.Rotate(new Vector3(0, 30, 0) * Time.deltaTime);
        
        // Scale the GameObject up and down over time
        float scale = Mathf.PingPong(Time.time, 1);
        transform.localScale = new Vector3(scale, scale, scale);
    }
}
