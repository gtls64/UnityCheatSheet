UI

To update text in a UI element, use the Text.text property.

using UnityEngine.UI;

public Text myText;
myText.text = "Hello, World!";

Add a click event to a button using the Button.onClick.AddListener method.

public Button myButton;
myButton.onClick.AddListener(TaskOnClick);

void TaskOnClick()
{
    // Button was clicked
}

Add a value change event to a slider using the Slider.onValueChanged.AddListener method.

public Slider mySlider;
mySlider.onValueChanged.AddListener(delegate { ValueChangeCheck(); });

void ValueChangeCheck()
{
    // Slider value changed
}
