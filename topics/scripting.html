Scripting

A basic MonoBehaviour script in Unity contains Start and Update methods. Start is called once when the script instance is being loaded, and Update is called once per frame.

public class MyScript : MonoBehaviour
{
    void Start()
    {
        // Called once when the script instance is being loaded
    }

    void Update()
    {
        // Called once per frame
    }
}

You can access components attached to a GameObject using the GetComponent method.

Rigidbody rb = myObject.GetComponent<Rigidbody>();

To instantiate a new GameObject, use the Instantiate method.

Instantiate(myObject, new Vector3(0, 0, 0), Quaternion.identity);

To destroy a GameObject, use the Destroy method.

Destroy(myObject);
